# Project Skills

Add skills here that are specific to this project. Each skill is a directory containing a `SKILL.md` file.

## Structure

```
skills/
  my-skill/
    SKILL.md
    # optional: templates, examples, etc.
```

A `SKILL.md` should follow this format:

```markdown
# Skill Name

## When to use

## Steps

1. ...
2. ...

## Examples

## Notes
```
